Implementations of the CAS protocol operations handlers using JDOM.

This package provides implementations of the ProtocolHandler and ProtocolMappingStrategy for the CAS /serviceValidate, /proxyValidate, and /proxy operations using JDOM to parse the XML response from the CAS server.
