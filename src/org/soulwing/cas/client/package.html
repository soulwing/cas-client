The CAS client API.

This package provides the classes needed to use all of the authentication services of a CAS server. Clients of this package should generally utilize the Validator interface for interaction with the CAS server. Clients should obtain a instance of Validator using the ValidatorFactory object. The ValidatorFactory requires an instance of UrlGenerator that will be injected into Validator instances created by it. The default implementation of UrlGenerator is SimpleUrlGenerator. This implementation must be configured with a ProtocolConfiguration bean that specifies CAS configuration properties.
